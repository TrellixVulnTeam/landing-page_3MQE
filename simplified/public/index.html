Projects

  • HiveStats (Efficient SSH at UCB)
    This command-line utility helps students navigate the instructional servers (i.e. hiveXX@cs.berkeley.edu) in the EECS department at UC Berkeley. I wrote this in a few afternoons over winter break during my junior year. The tool automatically logs students into the least-busy instructional server via SSH, saving student time when CPU-intensive course projects are assigned.
  • BachChain
    This project explored an application of blockchain technology. Built on Bluemix as part of BlueHack 2017, my housemates and I created a democratized music-sharing platform secured though smart-contracts and an mp3 hashing function. Here is a YouTube video we made documenting the project, and the code on Github.
